# コントリビューションガイド

## 参加条件

本プロトコルへのコントリビューションに際して、以下の条件に同意をお願いします。

### RF（ロイヤルティフリー）特許ポリシー

本プロトコルの仕様を実装するために**必須な技術（essential patent）**を保有する、
または将来保有する可能性があるコントリビューターは、その特許を世界中の誰に対しても
**ロイヤルティフリー（無償）でライセンス**することに同意します。

これは W3C の RF 特許ポリシー（https://www.w3.org/Consortium/Patent-Policy/）に準拠します。

### 許容される特許

上記の制約は「プロトコル必須特許」にのみ適用されます。
コントリビューターが独自に開発した**実装方式・派生技術**については、
各コントリビューターが自由に特許取得できます。

## 仕様変更の提案

1. GitHub Issues にて提案を投稿してください
2. 変更の背景・根拠を明記してください
3. 既存の実装への影響を記述してください
4. オープンな議論を経て合意形成します
