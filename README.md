# 感謝経済プロトコル（Gift Economy Protocol）

**Published v0.1（2026-03-08）**

---

## このプロジェクトについて

感謝経済プロトコルは、人と人の間に生じる感謝の行為をデジタルに記録・流通させ、それを経済的・社会的価値として可視化するための**オープンプロトコル**です。

贈与経済 2.0 のコンセプトに基づき、特定の企業・プラットフォームに依存せず、W3C DID / VC 等の国際標準に準拠した形で設計されています。

---

## コアコンセプト

- **感謝の2層構造**: 社会的事実（Layer 1 / パブリック）とメッセージ内容（Layer 2 / プライベート）を分離
- **本人コントロール**: 感謝のメッセージ内容は送信者が所有し、受取人の同意のもとで公開
- **トークン**: 感謝をデジタルトークンとして表現・移転
- **AI対話インターフェース**: 自然言語での感謝記録をAIエージェントがサポート
- **DID / VC 統合**: W3C 標準に準拠した分散型アイデンティティと検証可能な資格情報
